File Paths

Absolute Path

An absolute path is the complete path to a file or directory from the root of the file system.

/home/user/documents/file.txt

Relative Path

A relative path is a path that starts from the current working directory.

This is another sentence that I am writing.

documents/file.txt

Path with Special Characters

Paths can include special characters like spaces, which need to be handled properly.

/home/user/my documents/file.txt
